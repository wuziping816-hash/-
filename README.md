# 八字神煞逆向推理引擎

输入一组神煞条件，反推所有可能的八字干支组合。

**核心发现**：在规则一致的前提下，八字的完整神煞组合具有高度唯一性，召回率与唯一确定率均达 **99.20%**，超过 48 种神煞时可实现 **100%** 唯一识别，堪称“八字指纹”。

## 快速开始

1. 确保已安装 Python 3.8+
2. 下载或克隆本项目
3. 在项目目录下运行：

```bash
python 神煞倒查八字.py
```

## 使用方法

运行后进入交互式命令行，按以下格式输入条件：

```
输入神煞条件（如：日柱魁罡 年柱桃花），可选加 --strict 启用排他模式，输入 'run' 开始求解，输入 'exit' 退出：
>> 日柱魁罡
>> 年柱华盖
>> run
```

引擎将列出所有满足条件的八字干支。

### 条件格式

每条条件格式为：`柱位+神煞名`，柱位可选：
- `年柱`
- `月柱`
- `日柱`
- `时柱`

例如：
- `日柱禄神`
- `年柱桃花`
- `月柱天乙贵人`

可在末尾加 `--strict` 启用严格模式，确保候选八字恰好只有指定的神煞。
